markovclick
=============

``markovclick`` is a Python package for building higher order Markov chains from clickstream data, 
which can be used to analyse user pathways through a website and predict / identify common routes.

Installation
=============
To install ``markovclick``, first clone the repository::

    git clone https://github.com/ismailuddin/markovclick.git

And then install the package::

    python setup.py install
